i3Plus+ is a custom firmware for the Wanhao Duplicator i3 Plus. It offers a complete redesign of the touchscreen interface and new features.
Those features include:
  • Complete menu rework
  • Overhauled leveling assistant
  • Overhauled move menu
  • Live temperature graphs
  • PID Autotuning menu
  • Multiple preheat profiles with customizable temperatures
  • Printing statistics
It is also based on the latest Marlin firmware 1.1.0 instead of an old one like the original version.
